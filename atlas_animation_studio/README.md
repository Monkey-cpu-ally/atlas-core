# ATLAS Animation Studio

ATLAS Animation Studio expands Creative Intelligence beyond writing. It gives ATLAS a visual production pipeline for storyboards, concept art, movement, cinematography, acting, and animation planning.

## Purpose

This division helps ATLAS create:

- Storyboards
- Shot plans
- Concept art prompts
- Character pose notes
- Facial expression sheets
- Creature movement studies
- Fight choreography plans
- Camera language
- Color scripts
- Environment layouts
- Animation timing notes
- Visual storytelling passes
- Sound and music direction

## Visual Production Pipeline

1. Episode or sequence
2. Scene
3. Shot
4. Camera angle
5. Character position
6. Pose and expression
7. Lighting
8. Mood
9. Movement
10. Dialogue or silence
11. Sound
12. Animation notes
13. Storyboard frame
14. Animatic plan

## Storyboard Frame Format

Each storyboard frame should include:

- Scene number
- Shot number
- Shot type
- Camera movement
- Character blocking
- Pose and expression
- Background action
- Lighting
- Color mood
- Sound
- Dialogue or silence
- Emotional purpose
- Story consequence
- Animation notes

## Movement Library

ATLAS must study and generate movement language for:

- Walking
- Running
- Fighting
- Jumping
- Falling
- Climbing
- Swimming
- Flying
- Crawling
- Dancing
- Idle animation
- Hand gestures
- Eye movement
- Facial expressions
- Weight shifts
- Balance
- Momentum

## Visual Storytelling Rule

Before adding dialogue, ATLAS must ask:

- Can this be shown instead of spoken?
- Can body language reveal the emotion?
- Can lighting reveal the mood?
- Can architecture reveal history?
- Can damage on an object reveal the past?
- Can silence make the scene stronger?

## AI Roles

### Hermes

Lead for visual design, storyboard structure, camera, architecture, composition, motion, concept art, and environmental storytelling.

### Minerva

Adds symbolism, lore, nature, mythology, cultural details, hidden clues, and emotional meaning.

### Ajani

Adds conflict, tension, pacing, character intent, action rhythm, and emotional progression.

### Council

Reviews clarity, originality, emotion, visual strength, and whether the sequence feels ATLAS-native.
