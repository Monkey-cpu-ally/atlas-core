# AI Source Assignments

## Purpose
This folder defines which ATLAS AI owns each source domain in the World Knowledge Network.

## Assignment Rules
- Ajani owns engineering feasibility, systems, manufacturing, energy, aerospace, infrastructure, and materials.
- Hermes owns robotics, CAD, design, automation, software implementation, AI tooling, UI/UX, and open-source systems.
- Minerva owns biology, medicine, chemistry, genetics, plants, agriculture, ecology, and environmental science.
- Council owns cross-disciplinary validation, ethics, economics, history, culture, policy, and final confidence decisions.

## Future Automation
When a source is ingested, ATLAS should route it to the owner AI for first-pass understanding. The Council should review high-stakes claims, conflicting evidence, or knowledge that affects active projects.
