# World Sources Citations

## Purpose
This folder documents citation rules for ATLAS knowledge records.

## Citation Requirements
Every knowledge record derived from external sources must keep:
- source_id
- original_url
- title
- author or organization when available
- publication date when available
- date learned
- last verified
- confidence score
- AI owner

## Source Return Rule
ATLAS must always be able to return to the original source. If a source URL changes, ATLAS should keep the old URL and add a replacement URL rather than overwriting the history.

## Future Automation
The Knowledge Network should periodically validate links, flag dead sources, and request Council review when evidence changes or source trust drops.
