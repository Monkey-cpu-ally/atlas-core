# Countries and Regions

## Purpose
This folder organizes World Knowledge Network sources by country and region so ATLAS can learn from global research rather than only U.S.-based sources.

## Region Strategy
ATLAS should intentionally include sources from:
- North America
- Europe
- Asia
- Africa
- South America
- Oceania
- Middle East
- Global/international organizations

## Future Automation
Source records should preserve original country, region, and language. When ATLAS summarizes non-English material, it should preserve the original citation and mark whether a translation was used.
