# Subject Index

## Purpose
This folder maps World Knowledge Network sources to ATLAS subject domains.

## Subject Strategy
Sources may belong to more than one subject. For example, NASA may belong to aerospace, robotics, engineering, climate, and materials. ATLAS should route the same source to multiple AIs when appropriate.

## Future Automation
Each ingested knowledge record should receive subject tags and project links so ATLAS can reuse it during research, teaching, blueprint planning, and digital twin work.
