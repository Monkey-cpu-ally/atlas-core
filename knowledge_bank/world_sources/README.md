# ATLAS World Knowledge Network

## Purpose
The World Knowledge Network is the approved global source layer for ATLAS. It stores source records, trust rules, AI ownership, citation requirements, sync history, and future ingestion notes.

ATLAS should not blindly copy source material. ATLAS should read, understand, compare, summarize in its own words, preserve citations, and update knowledge when sources change.

## Core Principle
ATLAS does not collect information. ATLAS develops understanding. Every conclusion should be traceable to evidence, open to revision, and connected to its original sources.

## Knowledge Record Rule
Each learned item should store:
- title
- source_id
- original_url
- country
- region
- language
- ai_owner
- subject_tags
- summary_in_own_words
- key_findings
- evidence_notes
- confidence_score
- related_projects
- date_learned
- last_verified
- next_review

## Future Automation Notes
Future ATLAS jobs should check approved sources on a schedule, extract metadata, summarize findings, compare with existing knowledge, update confidence scores, and preserve a version history for every major knowledge record.
