# ATLAS Innovation Lab — Active Project Index

This file tracks the major ATLAS projects that should be reviewed, improved, scored, and advanced through the Innovation Lab.

Status meanings:

- Intake Needed — idea exists, but no full project intake yet.
- Discovery — researching existing technology and evidence.
- First-Principles Review — breaking into physics, engineering, biology, materials, and AI systems.
- Concept Evolution — creating conservative, advanced, luxury, industrial, sustainable, and future versions.
- Digital Twin Planning — preparing simulation requirements.
- Prototype Planning — preparing a real-world build path.
- Paused — not active right now.
- Archived — stored for later.
