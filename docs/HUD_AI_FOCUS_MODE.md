# ATLAS AI Focus Mode

## Locked visual behavior

The default ATLAS HUD remains the approved three-ring interface.

When the user selects an AI from the inner AI branch ring:

1. The middle and outer rings fade away.
2. The AI ring expands into one large transparent ring.
3. The large ring displays only the selected AI's working branches.
4. The center core remains the visual anchor.
5. Selecting **Back** or pressing **Escape** restores the original three-ring layout.

This mode is inspired by the approved single-large-ring reference, but it must remain visually consistent with ATLAS rather than copying the source artwork.

## Branch ownership

- **Ajani:** strategy, operations, business, economics, risk, leadership.
- **Minerva:** biology, chemistry, physics, medicine, nature, research.
- **Hermes:** engineering, robotics, software, manufacturing, architecture, systems.
- **Council:** review, synthesis, planning, building, validation, final decisions.

## Hidden-systems boundary

Knowledge banks, memory banks, graph memory, source registries, indexes, caches, and ingestion systems must not become permanent branch buttons. They remain backstage services used automatically by the AIs.

## Design law

> Evolve the brain. Preserve the face.

Focus Mode is a temporary transformation of the existing HUD, not a separate dashboard and not permission to add more permanent visual clutter.
