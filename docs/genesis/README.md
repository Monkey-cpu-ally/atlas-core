# ATLAS: Genesis

This directory contains the foundational rules for ATLAS Genesis v1.0.

## Documents

1. [ATLAS Constitution](ATLAS-CONSTITUTION.md) — mission, pillars, truth, privacy, and long-term acceptance rules.
2. [ATLAS Design Bible](ATLAS-DESIGN-BIBLE.md) — Hub states, Constellation Wheel, one-face presence rule, visual materials, colors, art, and motion.
3. [AI Personality Bible](AI-PERSONALITY-BIBLE.md) — Ajani, Minerva, Hermes, Council, speech, expertise, handoffs, and multilingual identity.
4. [Pulse and Awareness](PULSE-AND-AWARENESS.md) — news, weather, disasters, markets, events, everyday awareness, negotiation support, and consent rules.

## Genesis v1.0 Target

- Three distinct AI personalities.
- Voice-first idle and conversation behavior.
- One AI portrait at a time; three only for Council or explicit group request.
- Constellation Wheel matching the approved circular reference layout.
- Original graffiti-inspired project artwork inside a precise holographic technical interface.
- Live React and Godot visual connection through ATLAS Core.
- The Pulse personalized intelligence briefing.
- Consent-based awareness modes.
- Component-driven implementation for React, Godot, Rive, TouchDesigner, Tauri, and Unreal.

## Governance

These documents are design authority for Genesis. Code that conflicts with them should be revised or documented as a deliberate proposal before merge.
