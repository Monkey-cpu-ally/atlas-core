# ATLAS Project Bank

## Purpose
The Project Bank stores every active, paused, completed, and future ATLAS project in one organized place.

## AI Owner
Ajani — Chief Engineer.

## Supporting AIs
- Hermes — Chief Robotics & Design: robotics, design, HUD, CAD, automation.
- Minerva — Chief Scientist: science validation, biology, chemistry, ecology, medicine.
- Council — Executive Board: project approval, priorities, tradeoffs, and final direction.

## Allowed Content
- Project briefs
- Mission statements
- Requirements
- Research links
- Blueprint references
- Simulation notes
- Budget and timeline notes
- Risks and open questions
- Prototype milestones

## Future Automation Notes
ATLAS should create project records automatically when the user starts a major idea. Each project should connect to Memory Bank, Knowledge Bank, Blueprint Bank, Research Bank, and Digital Twin records.
