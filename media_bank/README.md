# ATLAS Media Bank

## Purpose
The Media Bank stores approved media references used by ATLAS, including images, videos, scans, audio notes, screenshots, design references, and visual inspiration metadata.

## AI Owner
Hermes — Chief Robotics & Design.

## Supporting AIs
- Ajani — Chief Engineer: scans, diagrams, parts, mechanisms, build references.
- Minerva — Chief Scientist: plant, animal, biological, environmental, and medical visual references.
- Council — Executive Board: permission rules and privacy review.

## Allowed Content
- User-approved images
- Screenshots
- Video reference notes
- Audio notes
- 3D scans
- Design reference metadata
- Visual tags and source links

## Future Automation Notes
ATLAS should only ingest private media with user permission. It should store descriptions, tags, relationships, and source links rather than blindly duplicating files.
