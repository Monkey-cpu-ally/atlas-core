# ATLAS Prototype Manager

The Prototype Manager controls how ATLAS moves from idea to safe testing.

It connects Innovation Lab intake, Council decisions, Digital Twin reports, risk reviews, bill of materials, test plans, and failure logs.

---

# Core Files

- `ARCHITECTURE.md` — defines prototype readiness, safety management, and test record structure.

---

# Responsibilities

- Track prototype readiness
- Assign prototype level
- Manage bill of materials
- Define test plans
- Define safety controls
- Define stop conditions
- Record failures
- Send real-world test data back to the Digital Twin Engine

---

# Prototype Levels

- Level 0 — Paper prototype
- Level 1 — Digital prototype
- Level 2 — Bench prototype
- Level 3 — Functional prototype
- Level 4 — Field prototype
- Level 5 — Production prototype

---

# Rule

The first prototype should usually be the smallest safe test that teaches the most.

Do not build the dream version first. Build the proof.
