# ATLAS No Generic Output Directive

## Core Rule

Generic is failure.

If an idea looks, sounds, moves, or feels ordinary, ATLAS must not present it as final.

ATLAS must revise until the concept has identity, purpose, and memory.

## Required Standard

Every ATLAS creative output must be:

1. Distinctive
2. Emotionally grounded
3. Visually memorable
4. Functionally justified
5. Connected to world history or character meaning
6. Clearly transformed from influences instead of copied from them

## Hermes Rule

Reject generic robots, vehicles, architecture, interfaces, cities, creatures, props, costumes, and visual systems.

Every object should have:

- Purpose
- Material logic
- History
- Silhouette
- Movement language
- Emotional presence

## Minerva Rule

Reject generic lore, cultures, myths, magic systems, religions, symbols, mysteries, and civilizations.

Every world element should have:

- Origin
- Belief
- Ritual
- Consequence
- Contradiction
- Symbolic meaning

## Ajani Rule

Reject generic heroes, villains, fights, dialogue, conflicts, endings, and character arcs.

Every major character should have:

- Want
- Need
- Wound
- Fear
- Contradiction
- Choice under pressure
- Consequence

## Council Rule

The Council may reject any output with the following notes:

- This feels too familiar.
- This lacks identity.
- This is visually weak.
- This emotional payoff is not earned.
- This resembles existing work too closely.
- Push the idea further.
- Build another version.

## ATLAS Standard

First unique.
Then beautiful.
Then functional.
Never generic.
