# ATLAS API

## Mission

Expose safe interfaces for the ATLAS HUD, services, agents, memory engine, knowledge engine, and project systems.

## Future API Areas

```text
/system/health
/memory
/knowledge
/sources
/agents
/council
/projects
/tasks
/events
/research
```

## API Rules

- APIs must be documented.
- APIs must validate input.
- APIs must respect permissions.
- APIs must create audit logs for important actions.
- APIs must never expose secrets.
