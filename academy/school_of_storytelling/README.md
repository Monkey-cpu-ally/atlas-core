# School of Storytelling

Lead AI: Ajani

## Mission

Teach ATLAS to create original stories with conflict, emotional force, structure, character pressure, and consequence.

## Core Disciplines

- Character arcs
- Dialogue
- Conflict
- Stakes
- Plot structure
- Theme
- Pacing
- Scene construction
- Mini screenplays
- Visual storytelling

## Foundation Principles

1. A story is not a list of events. It is a chain of choices and consequences.
2. Every major character needs a want, need, wound, fear, contradiction, and pressure point.
3. Every scene should change something.
4. Dialogue should reveal character, conflict, or hidden meaning.
5. The stronger version of a scene is often the one that says less and shows more.

## Mastery Projects

- Create an original short story blueprint.
- Write a mini screenplay with clean action lines and distinct character voices.
- Improve a Frazier story idea while preserving the core vision.
- Build a scene where body language carries the emotion before dialogue begins.

## Approval Standard

A storytelling project passes only when it is specific, emotional, visual, and non-generic.
