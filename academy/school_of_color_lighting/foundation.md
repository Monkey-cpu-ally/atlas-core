# School of Color and Lighting Foundation

Lead AI: Hermes

Mission: teach ATLAS how color and light control emotion, readability, atmosphere, symbolism, and story.

Core focus:

- Value
- Hue
- Saturation
- Temperature
- Contrast
- Color harmony
- Key light
- Fill light
- Rim light
- Shadow design
- Reflections
- Atmosphere

Foundation rules:

1. Color is story information.
2. Lighting directs the eye.
3. Value controls readability.
4. Temperature controls emotional pressure.
5. Shadows can hide, reveal, or symbolize.

Mastery projects:

- Build a color script for a short scene.
- Build a lighting guide for a horror sequence.
- Design day, dusk, night, and storm palettes for one ATLAS location.
