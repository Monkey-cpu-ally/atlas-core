# ATLAS Academy Labs

Schools teach. Labs experiment.

The Academy Labs are where Ajani, Minerva, Hermes, and the Council test ideas, compare versions, critique weak choices, and create original ATLAS applications.

## Lab Rules

1. Every lab experiment must have a question.
2. Every experiment must produce at least two versions.
3. The first version is never final.
4. The Council must identify weaknesses before approval.
5. The final version must be more specific than the starting idea.

## Initial Labs

- Story Lab
- Design Lab
- Motion Lab
- Cinema Lab
- Color Lab
- Character Lab
- World Lab

## Output Format

Each lab experiment should include:

- Experiment question
- Starting idea
- Constraints
- Version A
- Version B
- Critique
- Revision
- Final ATLAS version
- Lessons saved
