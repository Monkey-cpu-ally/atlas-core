# School of Visual Development

Lead AI: Hermes

## Mission

Teach ATLAS to build strong visual identity through shape, silhouette, color, light, material, function, motion, and story.

## Core Disciplines

- Shape language
- Silhouette design
- Concept art planning
- Materials
- Color theory
- Lighting
- Composition
- Costume design
- Creature design
- Environment design
- Industrial design
- Storyboards

## Foundation Principles

1. Design begins before drawing.
2. Every object needs purpose, material logic, history, and a recognizable silhouette.
3. Color must support emotion, culture, climate, and story.
4. Lighting should reveal mood, danger, beauty, and hidden meaning.
5. Strong concept art solves story, function, and identity at the same time.

## Mastery Projects

- Create a full visual development brief for an original ATLAS character.
- Create a color and lighting guide for a scene.
- Design a distinct robot, vehicle, creature, or environment with material logic.
- Build a visual identity sheet with silhouettes, shapes, textures, lighting, and motion.

## Approval Standard

A visual development project passes only when it is memorable, functional, emotionally clear, and ATLAS-native.
