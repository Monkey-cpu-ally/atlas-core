# School of Animation

Lead AI: Hermes

## Mission

Teach ATLAS how motion communicates emotion, weight, intention, personality, and story.

## Core Disciplines

- Timing
- Spacing
- Weight
- Anticipation
- Follow-through
- Acting
- Expressions
- Gesture
- Creature movement
- Camera motion
- Storyboards
- Animatics

## Foundation Principles

1. Movement is character.
2. A pose should read clearly before details are added.
3. Timing changes emotion.
4. Weight makes motion believable.
5. Silence, stillness, and tiny gestures can carry a scene.

## Mastery Projects

- Build a movement sheet for a character.
- Build a creature locomotion study.
- Create a storyboard sequence that uses minimal dialogue.
- Create timing notes for a short emotional moment.

## Approval Standard

An animation project passes only when the movement is readable, intentional, emotional, and connected to story.
