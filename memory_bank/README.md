# ATLAS Memory Bank

## Purpose
The Memory Bank stores ATLAS operational memory: conversations, project history, decisions, mistakes, successes, preferences, audit notes, and long-term AI context.

## AI Owner
Council — Executive Board.

## Supporting AIs
- Ajani — Chief Engineer: engineering decisions, build history, manufacturing lessons.
- Hermes — Chief Robotics & Design: robotics, design, HUD, CAD, and interface memory.
- Minerva — Chief Scientist: biology, chemistry, nature, medicine, and research memory.

## Allowed Content
- User preferences and standing instructions
- Project decisions and progress notes
- AI role memories
- Council decisions
- Lessons learned
- Failures, fixes, and audit logs
- Long-term ATLAS context

## Future Automation Notes
ATLAS should automatically write verified project decisions, AI learning summaries, and Council rulings into the Memory Bank. Memory entries should be searchable, tagged by AI owner, and connected to the Knowledge Bank when they contain reusable knowledge.
