# ATLAS Design Knowledge Bank

## Purpose
The Design Knowledge Bank is a core subsystem of the ATLAS Memory Bank. It stores structured knowledge about artists, architects, costume designers, industrial designers, engineers, and design principles.

## Core AI Roles
- Minerva: Research, color, materials, architecture, visual language.
- Ajani: Personal style, uniforms, branding, leadership aesthetics.
- Hermes: Robotics, vehicles, engineering, manufacturing, industrial design.

## Core Rule
Study principles and workflows from master creators. Do not copy copyrighted works. Synthesize original ATLAS designs from multiple influences.
