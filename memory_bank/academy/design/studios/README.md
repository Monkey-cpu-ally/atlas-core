# Studio Library

This library stores studies of studios, production teams, and design pipelines.

## Purpose
Studios teach ATLAS how large creative teams build a consistent visual world across many artists, departments, and media.

## Study Topics
- Production design
- Art direction
- Color scripting
- Character design
- Environment design
- Costume design
- Lighting
- Materials
- World-building systems
- Pipeline organization
- Style consistency

## Example Studios To Study
- Fortiche Production
- Riot Games
- Pixar
- Disney Animation
- DreamWorks
- Weta Workshop
- Industrial Light & Magic

## ATLAS Rule
Study workflow and cohesion. Do not clone a studio identity.
