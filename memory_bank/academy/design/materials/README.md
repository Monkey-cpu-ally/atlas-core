# Materials Library

The Materials Library teaches ATLAS how materials affect function, identity, durability, emotion, and manufacturing.

## Study Areas
- Metals
- Ceramics
- Glass
- Wood
- Leather
- Textiles
- Technical fabrics
- Carbon fiber
- Composites
- Bioplastics
- Smart materials
- Organic materials
- Recycled materials
- Surface finishes
- Wear, aging, and patina

## ATLAS Applications
- Robots
- Clothing
- Armor
- Tools
- Vehicles
- Architecture
- Furniture
- Product design
- HUD and interface hardware

## Core Rule
Material choices must serve function first, then reinforce the ATLAS design language.
