# Color Theory Library

The Color Theory Library teaches ATLAS how color communicates emotion, hierarchy, function, and identity.

## ATLAS Core Colors
- Ajani: crimson, strength, protection, leadership, tactical clarity.
- Minerva: teal, nature, learning, balance, living systems.
- Hermes: off-white, precision, engineering, clarity, construction.
- Council: purple, unity, judgment, shared intelligence.

## Study Areas
- Color harmony
- Value structure
- Temperature
- Contrast
- Emotional color scripting
- Lighting color
- Material color behavior
- UI accessibility
- Brand color systems

## ATLAS Applications
- HUD
- Logos
- Clothing
- Robots
- Vehicles
- Architecture
- Product families
- Warning and status systems

## Core Rule
Color must communicate meaning, not just decoration.
