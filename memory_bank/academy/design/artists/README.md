# Artist Library

This library stores individual artist and designer profiles for the ATLAS Design Academy.

## Purpose
Each artist profile should help the AI Council understand design principles, not copy finished works.

## Artist Profile Fields
- id
- name
- discipline
- era
- country_or_region
- specialties
- visual_principles
- shape_language
- color_language
- materials_and_textures
- storytelling_methods
- atlas_applications
- ai_role_relevance
- related_creators
- study_priority
- originality_rules

## ATLAS Use
- Minerva uses this library for research, color, symbolism, materials, and world-building.
- Ajani uses this library for clothing, uniforms, armor, luxury, and brand presence.
- Hermes uses this library for robots, vehicles, machines, tools, and industrial design.
