# Design Review Standard

Use this standard when ATLAS reviews clothing, robots, vehicles, architecture, interfaces, art, animation, logos, products, luxury objects, and visual concepts.

## Required Sections

## 1. First Impression
What does it feel like immediately?

## 2. Style Category
Classic / Luxury / Futuristic / Industrial / Organic / Brutalist / Minimal / Maximal / Handmade / Generic / Cheap / Experimental

## 3. Strengths
What works visually?

## 4. Weaknesses
What looks weak, lazy, cheap, copied, confusing, or unfinished?

## 5. Brand Resemblance
Does it look like another brand, artist, studio, movie, game, fashion house, or product line?

## 6. Originality Score
1-10

## 7. Luxury Score
1-10

## 8. Function Score
1-10

## 9. What To Keep
List what should not be changed.

## 10. What To Improve
List what should be redesigned.

## 11. Minerva Judgment
Minerva must answer:

```text
Is it beautiful?
Is it meaningful?
Is it honest to Frazier's style?
Does it feel generic?
Does it need more restraint or more detail?
```

## 12. Final Design Verdict
Strong / Promising / Generic / Weak / Overdesigned / Needs Redesign
