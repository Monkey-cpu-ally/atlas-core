# Public Digital Libraries

These sources provide legal, public, or access-controlled books, historical texts, audiobooks, and archived material.

## Project Gutenberg

Source Type: Public Domain Book Library
Trust Tier: Tier B for public-domain texts, Tier C for historical scientific material that may be outdated
Primary Banks: History, Philosophy, Storytelling, Literature, Mathematics, Science History
Use Case: Classic books, historical documents, philosophy, public-domain literature, old science texts.
Rule: Older scientific works must be marked as historical unless verified by modern sources.

## Standard Ebooks

Source Type: Curated Public Domain Ebook Library
Trust Tier: Tier B
Primary Banks: Storytelling, Philosophy, History, Art & Design
Use Case: Cleanly formatted classic public-domain books.
Rule: Use for readable versions of public-domain works.

## Wikisource

Source Type: Public Text Archive
Trust Tier: Tier B / C depending on document and provenance
Primary Banks: History, Philosophy, Law, Literature, Political History
Use Case: Original texts, historical documents, speeches, public-domain works.
Rule: Verify translation and edition when accuracy matters.

## LibriVox

Source Type: Public Domain Audiobook Library
Trust Tier: Tier C
Primary Banks: Storytelling, History, Philosophy
Use Case: Listening versions of public-domain works.
Rule: Use as companion to verified text sources, not as sole citation for precise wording.

## Internet Archive

Source Type: Digital Archive
Trust Tier: Tier B / C depending on item
Primary Banks: All banks
Use Case: Books, old manuals, videos, software history, scans, archived documents.
Rule: Check copyright/access status and source quality per item.

## Open Library

Source Type: Book Catalog and Lending Library
Trust Tier: Tier B / C depending on item
Primary Banks: All banks
Use Case: Book discovery, metadata, legally available lending where supported.
Rule: Treat as catalog/access point; evaluate individual books separately.

## Library of Congress Digital Collections

Source Type: National Archive
Trust Tier: Tier A / B
Primary Banks: History, Art & Design, Architecture, Culture, Law, Political History
Use Case: Historical documents, maps, images, recordings, manuscripts.
Rule: Strong source for historical artifacts and primary documents.

## HathiTrust Digital Library

Source Type: Academic Digital Library
Trust Tier: Tier B
Primary Banks: History, Science History, Literature, Philosophy, Architecture
Use Case: Digitized books and academic library collections.
Rule: Check access restrictions and item status.

## Europeana

Source Type: European Cultural Archive
Trust Tier: Tier B
Primary Banks: History, Art & Design, Architecture, Culture
Use Case: European museum, archive, and library collections.
Rule: Use for historical and cultural materials; verify rights and metadata.

## Digital Public Library of America

Source Type: Public Digital Archive
Trust Tier: Tier B
Primary Banks: History, Art & Design, Culture, Education
Use Case: American archives, images, documents, collections.
Rule: Use as discovery and archive source.
