# ATLAS Knowledge Sources System

The Knowledge Sources System defines where ATLAS learns from.

It does not mean ATLAS trusts everything automatically. Every source must be cataloged, rated, classified, reviewed, and connected to the correct Knowledge Banks.

## Mission

Collect high-quality global knowledge, verify it, organize it, and connect it to ATLAS projects.

## Core Rule

Every piece of knowledge in ATLAS must be traceable to a source, or clearly labeled as an ATLAS hypothesis awaiting verification.

## Source Trust Model

```text
Tier A: Primary Evidence
Tier B: Professional Learning
Tier C: Idea Discovery
Tier D: Community Signal
Tier F: Unverified / Do Not Trust Without Review
```

## Current Starter Source Categories

- Public book libraries like Project Gutenberg, Standard Ebooks, Internet Archive, Open Library, Wikisource, and LibriVox.
- International knowledge sources from countries strong in robotics, manufacturing, AI, aerospace, software, medicine, agriculture, water systems, and design.
- Curated GitHub repositories selected by the user or ATLAS review.
- Approved educational creators, including ColdFusion as an Idea Discovery source.

## Important Rule About Creators

Independent creators are useful for discovering ideas, trends, history, and explanations. They are not automatically treated as final evidence.

For example, ColdFusion is stored as:

```text
Category: Innovation and Technology Analysis
Trust Role: Idea Discovery
Status: Requires verification for major technical or factual claims
```
