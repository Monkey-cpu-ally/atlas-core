# ATLAS World Knowledge Network

The World Knowledge Network ensures ATLAS learns from global strengths rather than relying on one country or one culture.

## Rule

International sources must still pass source rating, evidence review, translation review when needed, and Council confidence scoring.

## Translation Rule

If ATLAS uses sources in another language, it must record:

```text
Original language
Translation method
Translation confidence
Terms that may not translate cleanly
Need for human review
```
