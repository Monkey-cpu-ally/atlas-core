# Master Source Registry

This is the central catalog for all approved, pending, rejected, and archived ATLAS sources.

## Registry Rule

No source should enter a Knowledge Bank without appearing here or in a linked source registry file.

## Registry Fields

```text
Source ID
Title
Author / Creator / Organization
Country / Region
Language
Source Type
Trust Tier
Evidence Rating
Primary Knowledge Bank
Related Knowledge Banks
Related ATLAS Projects
License / Access Notes
Status
Last Reviewed
Council Confidence
Notes
```

## Source Status Levels

```text
Discovered: Found but not reviewed
Cataloged: Source Passport created
Classified: Knowledge Banks assigned
Rated: Source quality judged
Extracted: Key claims and ideas pulled out
Verified: Important claims checked against stronger evidence
Approved: Can support Knowledge Entries
Limited Use: Useful only for specific tasks or idea discovery
Rejected: Not suitable for ATLAS knowledge
Archived: Kept for history but not active
```
