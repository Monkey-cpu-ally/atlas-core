# Approved Creator Source Registry

Independent creators can be valuable sources for ideas, explanations, trends, history, and demonstrations.

They are not automatically treated as primary evidence. Technical, medical, legal, financial, or scientific claims must be verified against stronger sources.

## Creator Trust Roles

```text
Primary Educator: Strong expert teaching source, still verified when needed
Technical Demonstrator: Useful for visual process and build examples
Idea Discovery: Useful for finding topics worth investigating
Historical Explainer: Useful for background, must verify primary claims
Community Signal: Useful for trends, weak evidence by itself
```
