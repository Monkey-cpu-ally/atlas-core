# Source Seed Plan For 22 Knowledge Banks

This file defines the first source categories each Knowledge Bank should collect.

## Goal

Each bank should begin with 50-100 carefully chosen sources before large-scale ingestion.

## 1. Artificial Intelligence

Seed sources:
- Official AI framework documentation
- University ML courses
- Peer-reviewed AI papers
- Model cards
- AI safety research
- Open-source AI repositories
- Conference talks
