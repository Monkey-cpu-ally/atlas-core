# Source Gap Engine

The Source Gap Engine identifies what sources ATLAS is missing.

## Mission

Do not only track what ATLAS has. Track what ATLAS lacks.

## Gap Types

```text
Missing source type
Weak evidence
Outdated source
Missing country / region
Missing language
Missing standards source
Missing manufacturer documentation
Missing patent research
Missing primary source
Overreliance on creators
Overreliance on one country
Weak project coverage
```

## Example Gap Report

```text
Knowledge Bank: Botany
Problem: Many general plant sources, not enough medicinal plant evidence reviews
Risk: Minerva may overvalue traditional claims without clinical support
Action: Add evidence reviews, pharmacology sources, toxicity references, and agricultural databases
Priority: High
```

## Project-Based Gap Report

```text
Project: Power Cell
Strong Banks: Physics, Chemistry, Electrical Engineering
Weak Banks: Manufacturing, Safety Standards, Economics
Action: Acquire manufacturing process sources, electrical safety standards, and cost analysis references
```

## Country Gap Report

```text
Knowledge Bank: Robotics
Current Countries: USA, Japan, Germany
Missing: Switzerland, South Korea, China, Singapore
Action: Add robotics lab sources, manufacturer docs, and government/academic research from missing regions
```

## Gap Priority

```text
Critical: Blocks project safety or feasibility
High: Needed for real engineering or evidence quality
Medium: Improves knowledge depth
Low: Nice to have
```

## Council Questions

The Council should ask:

```text
What do we not know?
What sources are missing?
Which project is affected?
What risk does the gap create?
What source type should be acquired next?
Who should review it?
```
