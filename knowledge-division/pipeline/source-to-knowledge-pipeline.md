# Source-to-Knowledge Pipeline

This pipeline explains how a source becomes usable ATLAS knowledge.

## Pipeline Overview

```text
Discovery
↓
Source Passport
↓
Classification
↓
Source Rating
↓
Extraction
↓
Verification
↓
AI Review
↓
Council Review
↓
Knowledge Bank Entry
↓
Project Link
↓
Memory Record
```

## 1. Discovery

A source is found from:

- Public digital libraries
- Academic sources
- Government sources
- Standards
- Patents
- Manufacturer documentation
- GitHub
- Approved creators
- User notes
- ATLAS research

Status: Discovered

## 2. Source Passport

A Source Passport is created before the source is studied.

Required:

```text
Title
Author / Creator / Organization
Country
Language
Source type
Trust tier
Evidence rating
Knowledge banks
License / access notes
```

Status: Cataloged

## 3. Classification

ATLAS assigns:

```text
Primary Knowledge Bank
Related Knowledge Banks
Assigned AI
Related ATLAS projects
Source category
```

Status: Classified

## 4. Source Rating

ATLAS rates source quality.

```text
A-Level: Primary evidence
B-Level: Strong learning source
C-Level: Useful but needs verification
D-Level: Weak source
F-Level: Do not trust without review
```

Status: Rated

## 5. Extraction

ATLAS extracts:

- Main ideas
- Definitions
- Methods
- Data
- Claims
- Limitations
- Safety notes
- Design ideas
- Project relevance

Status: Extracted

## 6. Verification

ATLAS checks whether important claims are supported by stronger evidence.

Questions:

```text
Is the source current?
Is it supported by stronger sources?
Does it conflict with verified knowledge?
Is it opinion, marketing, or tested evidence?
Does it need expert review?
```

Status options:

```text
Verified
Partially Verified
Needs More Evidence
Rejected
Archived
```

## 7. AI Review

Hermes checks engineering and build relevance.

Minerva checks scientific, biological, environmental, design, and story relevance.

Ajani checks strategy, risk, business, economics, and planning relevance.

Status: AI Reviewed

## 8. Council Review

The Council decides:

```text
Store as verified knowledge
Store as research note
Store as idea discovery
Connect to project
Archive only
Reject
```

Status: Council Reviewed

## 9. Knowledge Bank Entry

The source becomes a Knowledge Entry or supports an existing entry.

Required links:

```text
Source Passport
Knowledge Bank
Evidence rating
Related topics
Related projects
Review date
```

Status: Stored

## 10. Project Link

If useful, the source is linked to ATLAS projects.

Examples:

- Weaver
- Green Robots
- Power Cell
- Metal Flowers
- ATLAS HUD
- Digital Twin Engine
- Luxury Design Intelligence
- Hyper Axel

Status: Applied

## 11. Memory Record

ATLAS records:

```text
What changed
Why it changed
Who reviewed it
What projects were affected
What needs future review
```

Status: Remembered

## Final Rule

No source becomes trusted ATLAS knowledge without a Source Passport, rating, review, and traceable connection to evidence.
