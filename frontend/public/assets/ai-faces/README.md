# ATLAS AI Portrait Assets

This folder is reserved for the approved Ajani, Hermes, and Minerva portrait assets.

## Approved reference

The approved reference sheet supplied by Frazier defines the canonical appearance of all three AIs:

- **Ajani:** braided copper-orange hair, facial scars/markings, gold earring, veteran-warrior presence.
- **Hermes:** black locs, branch/thorn crown, dark round glasses, relaxed trickster-engineer presence.
- **Minerva:** dark green/teal locs, warm expression, facial markings, scholar/guardian presence.

The original image should be added here as:

```text
atlas-ai-face-reference.jpg
```

Production exports should use transparent PNG or WebP files and follow this structure:

```text
ajani/
  neutral.webp
  thinking.webp
  speaking.webp
  serious.webp
  approval.webp

hermes/
  neutral.webp
  thinking.webp
  speaking.webp
  scheming.webp
  serious.webp

minerva/
  neutral.webp
  thinking.webp
  speaking.webp
  concern.webp
  guidance.webp
```

## Rules

- One face appears at a time during normal use.
- All three appear only in Council Mode or when explicitly requested.
- Portraits remain secondary to the user's project or task.
- No redesign may alter the defining face, hair, markings, clothing, or personality cues without Frazier's approval.
- Final animated versions may use Rive, Live2D, or sprite sequences, but must preserve this reference design.
- Portrait images should be center-framed, transparent, and optimized for phone, tablet, and desktop displays.

## Recommended export sizes

- Source master: 2048 × 2048 or larger.
- Desktop/tablet portrait: 1024 × 1024 WebP.
- Mobile portrait: 512 × 512 WebP.
- Thumbnail: 256 × 256 WebP.
